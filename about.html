üéØ Project Overview

The QA Training Site is a comprehensive web application designed specifically for automation testing with Playwright. This project provides a realistic testing environment with various web elements, forms, interactions, and scenarios that automation engineers encounter in real-world applications.

Built as a standalone HTML application with embedded CSS and JavaScript, it offers a complete testing playground without requiring complex setup or external dependencies. Each page is carefully crafted to include deterministic locators and comprehensive testing scenarios.

üß™ Testing Scenarios

Each page is designed with specific testing scenarios in mind, covering common automation challenges:

  • Element Locators: IDs, classes, attributes, text content, CSS selectors
  • Wait Strategies: Implicit waits, explicit waits, element visibility
  • Form Validation: Required fields, email format, password strength
  • Dynamic Elements: AJAX-loaded content, conditional rendering
  • User Interactions: Click, type, select, hover, drag & drop
  • Navigation: Page transitions, browser history, routing
  • File Operations: Upload, download, file handling
  • Dialog Handling: Alerts, confirms, prompts, modals

üìñ Usage Guidelines

This training site is designed to be used with Playwright automation framework. Each page includes deterministic locators and predictable behavior patterns.

Getting Started:

  • Navigate through different pages using the main navigation
  • Use browser developer tools to inspect element locators
  • Test various user interactions and workflows
  • Practice writing Playwright tests for different scenarios
  • Explore responsive design by resizing the browser window
